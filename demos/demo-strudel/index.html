Live-coding workshop: write a pattern, hear it, change it

This is Strudel, the browser version of the live-coding language TidalCycles. The code below is the instrument. Press the play control in the editor to start sound, change any value, then press play again to hear the edit. Press stop to silence it. Nothing is installed: this is the same kind of tool a performer uses at an algorave.

Try editing the code: paste one of these in, then press play

Make the beat busier (more complex, less danceable, an IDM move):

stack(
  s("bd*2 ~ sd [~ bd]"),
  s("hh*16").gain(0.4),
  note("c2 eb2 g1 c2").s("sawtooth").lpf(700)
)

Add a glitch: chop the hats into stutters and randomly speed up hits:

stack(
  s("bd ~ sd ~"),
  s("hh*8").chop(4).gain(0.5),
  s("bd sd").sometimesBy(0.4, fast(2))
)

Make a melody instead of a beat:

note("c4 e4 g4 b4 a4 g4 e4 c4")
  .s("triangle")
  .room(0.5)
  .slow(2)

Turn one line off: put // in front of any line inside the stack, then press play.

Strudel runs in your browser and loads its engine and drum samples over the network, so it works best on the hosted (https) page. If the editor does not load inside Canvas, open this workshop in a new tab ↗. Full language reference and more examples: strudel.cc.
